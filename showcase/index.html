Same prompt. Different depth.

"Design a SaaS pricing page for a developer tools product (3 tiers). Make it premium, not template-y."

Without Skill Baseline

Visual Strategy

"Asymmetric emphasis — Pro card physically dominates via spatial hierarchy." Cards on dark background, Pro gets gradient border.

Layout

Pro card is wider (380px vs 340px). Cards at 32px padding, 16px radius, 24px gap.

Toggle

Segmented control with sliding background. Price does vertical slide-fade (150ms). Provides full CSS + JS implementation.

Anti-Template Moves

  • Pro wider not taller
  • Gradient only on 1px border
  • CTA above features
  • Dark mode by default
What's missing: No explanation of WHY CTA goes above features. No accessibility (focus states, keyboard nav). No CLS prevention. No loading states. No escape hatch for equal-tier products.
With Skill polanyi-design

Design Thesis

Tacit insight: "Pro is the protagonist. Hobby and Enterprise are supporting cast that exist to make Pro's value obvious through contrast."

Layout + Rationale

Pro card at lg:scale-[1.02] with shadow-[0_24px_48px_-12px]. Price block has fixed h-[72px] to prevent layout shift on toggle.

Tacit insight: "The 'Most popular' badge uses absolute -top-3.5 — it breaks the card boundary intentionally. A badge inside padding is subsidiary; a badge that breaches is focal."

Toggle

Savings badge uses emerald not brand indigo.

Tacit insight: "Emerald encodes 'money saved' — a different semantic than brand identity. Using brand blue here would flatten the information hierarchy."

What NOT to Do

  • "Don't hover-lift pricing cards — they imply the card is clickable, but only the CTA is"
  • "Don't list 15+ features — group into 2-3 categories"
  • "Don't make all CTAs same visual weight"

Accessibility

focus-visible:outline-2 outline-offset-2 on all buttons. aria-hidden on decorative icons. role="list" on features.

Escape Hatch

Tacit insight: "If your product genuinely has equal adoption across tiers, flatten the hierarchy. The asymmetric strategy assumes Pro is the business model."
